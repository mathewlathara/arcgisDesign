DeX-Terity Tools

Data Platform / Descriptive

Access and view disparate sources of data using our map tool.

Graphical Analysis / Explanatory

Visualize changes to land-use, population, air and water quality by watershed over time. Explore causal links between these parameters.

Trends Analysis / Predictive

Use DeX-Terity forecasting tools to compare and assess environmental and societal risks arising from different watershed futures.

Decision-Making Tool / Prescriptive

Mitigate risks by identifying and optimizing watershed protection priorities and actions.
